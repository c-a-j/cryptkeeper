## Crypt Keeper

Yes, it's another password manager.

### The Basic Plan

Obviously, this project is its early stages. The main goal is to create a
password manager that can be used with git similar to `pass` and `gopass`, but
abstract the metadata by using an encrypted index.

### Near-term features

- Get the basics working
- Support individual configs for different crypts

### Medium-term features

- Migrate `pass` store to a crypt
- Migrate BitWarden items to a crypt
- QR Code gpg key backup

### Features far down the road

- Browser integration

### Features on the opposite coast

- Mobile and desktop applications
  - Import key from QR Code
  - Facial recognition to unlock
